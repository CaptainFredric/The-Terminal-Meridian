# Icons

Replace these placeholders before publishing to the Chrome Web Store.

Required sizes (all PNG, square, transparent background):

| File | Size | Used for |
| --- | --- | --- |
| `icon16.png` | 16x16 | Favicon / tab strip |
| `icon32.png` | 32x32 | Windows taskbar |
| `icon48.png` | 48x48 | Extensions management page |
| `icon128.png` | 128x128 | Chrome Web Store listing & install dialog |

Suggested style: a stylized "M" or compass-rose mark in `#6f8fff` (Meridian
accent blue) on a dark `#0a0e1a` panel, matching the terminal's visual
identity.

The placeholder `.png` files in this directory are tiny valid PNGs so Chrome
will load the unpacked extension without warnings — they are NOT real
artwork. Swap them out before any release build.
